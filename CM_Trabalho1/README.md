## Introdução
Este trabalho prático teve o propósito de introduzir a turma aos fundamentais da linguagem de programação Kotlin e o software Android Studio. As tarefas propostas envolvem desenvolver código na linguagem Kotlin e trabalhar com o Android Studio para criar uma aplicação móvel simples.

## Resumo do Sistema
O código desenvolvido segue as tarefas planeadas pelo enunciado fornecido, tendo em conta o trabalho na linguagem Kotlin focado na iteração, cálculo e geração de valores bem como o trabalho com classes e sub-classes relacionadas; enquanto que o trabalho em Android Studio descreve simplesmente o desenvolvimento básico de uma aplicação móvel.
Para a secção 2, o exercício 1 (exer_1.kt) requer 3 métodos diferentes para a inicialização de um vetor de 50 elementos de números quadrados perfeitos.
O exercício 2 (exer_2.kt) requer o desenvolvimento uma calculadora baseada no terminal de comandos com funcionalidades de adição, subtração, multiplicação e divisão de valores.
O exercício 3 (exer_3.kt) requer a geração de uma sequência de alturas para a simulação de uma bola que mantém apenas 60% da sua altura máxima por cada salto.

Para a secção 4, o programa desenvolvido requer as habilidades de desenvolvimento básico para trabalhar com o Android Studio e criar/construir uma dada aplicação móvel

Para a secção 6, o exercício (virtual_library.kt) requer o desenvolvimento de classes e sub-classes dedicadas aos objetos e funcionalidades de uma biblioteca virtual de modo a conseguir adicionar, alugar, devolver, demonstrar e procurar por autor cada livro que for implementado dentro do dado sistema.

## Arquitetura e Design
PASTAS:
- CM (Pasta principal de código para o trabalho em Kotlin)
- code (Pasta de código para Android Studio
- Hello_World.zip (Código de Android Studio extraído em formato .zip através do IntelliJ IDEA)

A estrutura das pastas do repositório foi organizada com a intenção de dividir o mesmo entre cada trabalho que for realizado durante o semestre.
## Implementação
A pasta CM contém 4 ficheiros principais: exer_1.kt, exer_2.kt, exer_3.kt e virtual_library.kt. Estes contém a resolução dos exercícios sugeridos nas secções 2 e 6 do enunciado de Tutorial 1.
A pasta Hello_World.zip (e da mesma forma a pasta code) contém a resolução da secção 4 do enunciado de Tutorial 1.

## Testes e Validação
Casos de teste  
exer_2.kt:
- Escolha de opções fora do menu disponibilizado.
- Introdução de números/caracteres inválidos para operações aritméticas.
- A divisão de um número por zero.

virtual_library.kt:
- A incialização da quantidade de cópias disponíveis de um livro como um valor inválido (zero ou menor).
- A inicialização do ano de publicação de um livro como um número não razoável (zero, menor que zero ou depois de 2026)

## Instruções de Uso
O conteúdo da pasta CM poderá ser aberto e configurado no software IntelliJ IDEA, enquanto que o conteúdo do zip Hello_World.zip poderá ser aberto e configurado no Android Studio.
